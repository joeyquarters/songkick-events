A plugin used to grab an artist's upcoming events and display them in a theme by easily employing a shortcode.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[songkick-events]` in your templates
